# StandUp[Alpha]

统计站立办公时间的小 App，提供简单的统计视图

![App 截图](screenshot.png)

## Roadmap

### Features

- [x] 季度视图 + 滚动
- [x] 简单统计
- [x] 用户设置
- [x] 久坐/久站提醒
- [x] 单日视图
- [x] Import / Export
- [x] 暗黑模式
- [ ] i18n

### Develop

- [x] Github CI
- [ ] Logger
- [ ] CLI Support


## Development

### Prerequisite

- [Tauri environment](https://tauri.app/v1/guides/getting-started/prerequisites/)
- node > 20
- pnpm
- cargo

### Run

```
$ git switch -c dev origin/dev
$ pnpm i
$ pnpm tauri dev
```
